# 🌌 NeonGlyph: Signal Synthesis & Terminal Projection Engine

> "The terminal is not a console. It's a phosphor-lit landscape."

**NeonGlyph** is an atmospheric rendering framework that transforms images, video, and live streams into cinematic glyph landscapes. It treats the terminal as a stage for signal synthesis, bridging the gap between retro-computing aesthetics and modern graphics engineering.

![NeonGlyph Header](https://via.placeholder.com/1200x400.png?text=NeonGlyph+v0.4.0+Terminal+Cinema)

## 📜 The NeonGlyph Principles

1. **Interpretation over Reproduction**: NeonGlyph does not attempt to reproduce images faithfully. It synthesizes signal atmospheres through glyph density, temporal persistence, and phosphor-inspired projection.
2. **Artifacts as Texture**: Jitter, tearing, and glitching are not errors. They are the visual vocabulary of the machine.
3. **The Phosphor Aesthetic**: Every projection should feel like it's emerging from a cathode-ray tube. Glow, decay, and scanlines are the foundations.
4. **Emotional Residue**: Through **Projection Memory™**, the engine allows past visual states to subtly linger as textual ghosts, creating a sense of history on the terminal landscape.

## ✨ Signature Features

- **🎭 Cinematic Adaptive Rendering™**: An intelligent layering system that dynamically blends **Braille detail** with **ASCII density**.
- **🎥 Live Neural Streaming**: Real-time webcam and video file streaming.
- **⏺️ ANSI Session Recording**: Capture live terminal performances to `.nglyph` files.
- **🌊 Signal Morphing**: Real-time interpolation between visual identities (e.g., `noir:cyberpunk`).
- **🧠 Projection Memory™**: Persistent "emotional residue" that accumulates textual ghosts over time.
- **⚡ Signal Corruption Shaders**: Aesthetic instability events (`jitter`, `tearing`, `glitch`).

## 🚀 Usage

### Launch Interactive Neural Performance
```bash
python main.py synthesize --webcam --interactive --preset cyberpunk --theme tokyo
```

### Signal Morphing with Emotional Residue
```bash
python main.py synthesize input.jpg --morph noir:cyberpunk --duration 12s --effect memory
```
